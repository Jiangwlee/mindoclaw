# MINDOCLAW

MindoClaw是一个7x24小时，完全本地化部署的工作助理。

## 目录结构

**文档命名规范**: `[编号0000-9999]-[文档名称]-[创建日期].md`

- `docs/`: 项目文档
    - `architeture/`: 架构设计
    - `brainstorms/`: 头脑风暴。当头脑风暴结束时，将讨论的主题、形成的结论落盘成文件。
    - `features/`: 功能文档。为每个feature创建子目录。
        - `feat-[功能编号]-[功能名称]/`
            - `design.md`: 功能设计文档
- `github/`: 参考的开源项目，比如：nanobot、pi-mino。此目录被`.gitignore`忽略，仅用于本地研究参考使用
